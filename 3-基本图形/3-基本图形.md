# 3-基本图形

## 简单的基本图形

* circle

  * cx cy r
  * fill stroke stroke-width transparent
  * style方式

* rect

  * width height x y
  * rx ry (设置长方形的圆角，只设置一个rx的话，相当于就是个圆，即rx和ry相等，不等的时候其实就是椭圆)

* line

  * x1 y1 x2 y2
  * stroke-opacity (0-1透明度)

* ellipse

  * 椭圆
  * cx cy rx ry

* polyline
  
  * 折线
  * points (设置多个点)

* polygon

  * 创建含有不少于三个边的图形
  * points (设置多个点)

* path

  * M = moveto
  * L = lineto
  * H = horizontal lineto
  * V = vertical lineto
  * C = curveto
  * S = smooth curveto
  * Q = quadratic Belzier curve
  * T = smooth quadratic Belzier curveto
  * A = elliptical Arc
  * Z = closepath
